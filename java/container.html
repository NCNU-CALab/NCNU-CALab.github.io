    容器類別

    對絕大部分應用來說，資料量的大小，必須等到執行期間才知道。

    由於 Java 的陣列一但產生後，就沒有辦法變更其大小，因此 JDK 在 java.util 裡面提供了相關的容器類別，比較常用的類別有：

    • Vector：可視為長度沒有限制的陣列
    • Hashtable：利用 get 和 put 函數把 (key, val) 放進該容器內
    • Enumeration：可利用此類別將 Hashtable 內的資料逐一取出
    • Stack with push() pop() methods
