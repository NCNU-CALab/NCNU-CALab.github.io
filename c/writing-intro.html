    程式語言

    C語言的結構

    • 註解 (comments)(給人看的)
    • 函數 (functions)
      • 函數名稱 (function name)
      • 參數 (parameters)
      • 區域變數宣告 (local variable declarations)
      • 敘述 (statements)

    敘述的組成

    • 常數、變數、運算符號、分號
    • if () {} else {}
    • while () {}
    • do {} while {}
    • switch () { case 1: case 2: case 3}
    • break
    • continue
    • goto

    Java 語言的結構

    • 註解 (comments)(給人看的)
    • 類別 (class)
      • 變數宣告 (variable declarations)
      • 函數宣告 (method declarations)
        • 函數名稱 (function name)
        • 參數 (parameters)
        • 區域變數宣告 (local variable declarations)
        • 敘述 (statements)
